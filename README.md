# Ease MD

A lightweight, Typora-style Markdown editor for Windows and macOS, built with Tauri.

## Getting started

This repository provides a minimal Tauri shell and a static `dist/` page to confirm the desktop wrapper boots.

### Prerequisites

- Rust (stable)
- Node.js (for future frontend tooling)
- Tauri CLI (`cargo install tauri-cli`)

### Run the app

```bash
cargo tauri dev
```
